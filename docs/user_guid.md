# 软件使用说明书（user_guid.md）

> 模块 5 交付物 · 面向最终用户的操作指南

## 1. 登录

1. 打开 http://localhost:5173。
2. 输入账号密码（管理员 admin/admin123，普通用户 viewer/viewer123）。
3. 点击「登录」，进入仪表盘。

> 管理员可执行采集、导出、任务管理；普通用户仅能查询与查看。

## 2. 仪表盘

进入后默认展示：
- 顶部统计卡片：股票标的数、基金标的数、日线行数、净值行数。
- 最近采集运行记录（触发方式、状态、影响行数、来源）。
- 最近导出记录。

## 3. 股票查询

1. 左侧菜单进入「股票查询」。
2. 选择股票（支持搜索），可选日期范围，点击「查询」。
3. 查看收盘价走势图与明细表格（涨跌幅红涨绿跌）。
4. 管理员可点击「采集该股票」实时补数，或「导出CSV」。

## 4. 基金查询

操作同股票查询，展示单位净值与累计净值双曲线及日增长率。

## 5. 采集任务

- **查看任务**：任务列表显示类型、目标代码、定时规则、启用状态。
- **新建任务（管理员）**：点击「新建任务」，填写名称、类型、目标代码（逗号分隔）、cron（留空仅手动）。
  - cron 示例：`0 18 * * 1-5` 表示工作日 18:00 自动采集。
- **执行任务（管理员）**：点击「执行」立即运行，完成后提示状态与行数。
- **查看日志**：点击「日志」查看该任务历史运行记录。
- **删除任务（管理员）**：点击「删除」并确认。

## 6. 数据导出

1. 进入「数据导出」页查看历史记录。
2. 管理员点击「新建导出」，选择数据集（股票日线/基金净值/标的列表）、格式（CSV/Excel/Parquet）、可选代码。
3. 导出成功后在列表点击「下载」获取文件。

## 7. 退出登录

点击右上角「退出登录」清除登录状态并返回登录页。

## 8. 演示建议流程（5–8 分钟）

1. 登录（admin）。
2. 仪表盘展示数据概览。
3. 股票查询 → 看图表 → 点「采集该股票」展示实时入库 → 数据刷新。
4. 任务页 → 新建一个定时任务 → 执行一次 → 看日志。
5. 导出基金净值 CSV → 下载打开。
6. 退出，用 viewer 登录，演示采集/导出按钮消失（权限控制）。
